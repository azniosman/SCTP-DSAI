## Lesson Objectives

Learners will understand:

- Data Architecture
  - Concepts
  - Principles
  - Types
- Object Storage

Learners will be able to:

- Consider trade-offs between different data architecture types.
- Perform create and read operations on Redis.
- Fetch data from Google Cloud Storage.
